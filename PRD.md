# 📑 D-Garden (도파민 가든) 제품 요구사항 정의서

## 1. 프로젝트 개요 (Overview)
- **제품명:** D-Garden (도파민 가든)
- **슬로건:** 당신의 업무 비트가 만드는 예술적 정원 (Ambient Mental Guardian)
- **핵심 가치:** 직장인의 실시간 PC 행동 패턴(타이핑 바이오리듬, 마우스)을 가볍고 명확한 알고리즘 수식으로 분석하여 멘탈 상태를 HTML5 Canvas 기반의 생성형 디지털 아트로 시각화하고, Supabase Realtime을 통해 관객과 시각적으로 소통하는 웰니스 대시보드.

## 2. 대상 사용자 및 유스케이스 (User & Use Cases)
- **메인 사용자:** 모니터 한구석에 D-Garden을 켜두고 업무 진행. 본인의 입력 속도와 오타율 등에 반응해 자라나는 식물을 보며 현재 몰입/과열 상태를 직관적으로 인지.
- **해커톤 관객 및 심사위원:** 발표 화면의 QR 코드를 스캔해 모바일 웹 리모컨에 접속. 실시간으로 응원 메시지나 키워드를 입력하여 발표 화면 속 식물에 파티클 이펙트를 발생시키고 점수에 영향을 줌.

## 3. 핵심 기능 요구사항 (Product Features)
### 3.1. PC 행동 패턴 수집 및 분석 엔진 (Behavior Tracking Engine)
- **분석 지표:**
  - CPM (Characters Per Minute): 최근 3초간의 분당 타수 계산.
  - 오타율 (Backspace Ratio): 전체 타이핑 중 Backspace 키가 차지하는 비율 계산.
- **경량 행동 알고리즘 반영:**
  - 타수가 높고(`CPM >= 300`) 오타율이 낮으면 ➔ **안정적 초몰입 상태**로 판정.
  - 타수가 매우 높고(`CPM >= 400`) 오타율이 높으면(`Backspace >= 20%`) ➔ 업무 과부하 및 스트레스로 인한 **과열(Rage) 상태**로 판정.

### 3.2. SDD 기반 멘탈 스코어 연산 규칙 (Mental Score Core)
- **시스템 원칙:** 0점부터 100점 사이의 단일 게이지 시스템 (초기값: 50점).
- **실시간 가중치 수식 (spec.json 명세화):**
  - `입력 없음 (Idle)`: 매초 -0.5점 자연 감소 (100초간 부재 시 0점 도달).
  - `마우스 무브/휠`: 매초 +0.5점 상승 (최대 55점 상한제한 - 리딩 상태 방어).
  - `간헐적 클릭/짧은 타자`: 건당 +1.0점 상승 (최대 60점 상한제한).
  - `행동 분석 엔진 감지 (초몰입/지속 타자)`: 매초 +1.0점 상승 (상한선 없음, 과열 모드 진입 가능).
- **관객 반응 반영:** 긍정 키워드 유입 시 +2점, 부정 키워드 유입 시 -2점. (단, 초당 최대 변동 폭은 +-5점으로 제한하는 스로틀링 밸브 적용).

### 3.3. HTML5 Canvas 기반 생성형 디지털 아트 식물 (Generative Art Engine)
- **4단계 식물 상태 매핑:**
  1. `0 ~ 20 [휴면 (Dormant)]`: 회색(Gray) 톤, 성장이 멈추고 생기를 잃은 정적 상태.
  2. `21 ~ 60 [최적 몰입 (Goldilocks)]`: 에메랄드 네온(Emerald Neon) 빛, 가장 파릇파릇하고 아름답게 가지를 뻗는 상태 (초기 상태).
  3. `61 ~ 90 [과열 (Overheat)]`: 보라색(Purple) 톤, 줄기가 거칠어지거나 불꽃 같은 파티클이 튀는 과열 상태.
  4. `91 ~ 100 [강제 쿨다운 (Cooldown)]`: 다크 네이비(Dark Navy) 톤, 화면이 차분해지며 사용자가 심호흡을 할 수 있도록 유도하는 정적인 상태.

### 3.4. Supabase Realtime 기반 관객 텍스트 인터랙션 (Crowd-Hack)
- **실시간 동기화:** 관객이 모바일 리모컨에서 응원 문구를 입력하면 Supabase Realtime 채널을 통해 대시보드로 즉시 라우팅 (0.1초 이내).
- **패턴 매칭 감성 분석:** `spec.json`에 정의된 긍정/부정 딕셔너리와 실시간 문자열 비교(String Matching)를 수행하여 즉각적으로 점수에 반영.
