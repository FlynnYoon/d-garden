# D-Garden Bridge

Cursor/VS Code에서 발생하는 타이핑을 추론해 로컬 SSE 서버(`http://127.0.0.1:7331/events`)로 브로드캐스트하는 확장.

D-Garden 페이지(`engine/editor-bridge.js`)가 이 엔드포인트를 구독해서, 에디터에서 코딩하는 동안에도 멘탈 스코어와 나무가 반응한다.

- 외부 네트워크 전송 없음 — 127.0.0.1 바인딩
- 키 내용은 절대 전송하지 않음 — 키 입력 **횟수**와 삭제 횟수만 집계
- 상태바에 연결 상태 표시 (`♥ D-Garden` = 페이지 연결됨)
